# 🛠️ Engineering Manual (`eng-manual`)

This repository is my **Primary Technical Reference**, **TIL (Today I Learned) Log**, and **Digital Garden**. It is a curated collection of distilled knowledge, production-ready snippets, and architectural notes from my daily engineering work.

## 📝 Project Philosophy
- **Living Document:** This is a dynamic resource that grows alongside my technical journey.
- **Showcase of Growth:** It tracks my evolution as an engineer and my commitment to continuous learning.
- **Open Source Knowledge:** While these are personal notes, they are shared publicly to help others and demonstrate my technical stack.

## 🚦 Note Maturity Scale
To balance speed and quality, I use a maturity system for my documentation:
- 🌱 **Seed:** Raw thoughts or quick snippets captured during active development (WIP).
- 🌿 **Sapling:** Structured notes with verified code examples and explanations.
- 🌳 **Evergreen:** Polished, production-standard documentation intended for long-term reference.
